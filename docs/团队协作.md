## 2. Setup Guide for Team Members

### Prerequisites

- Install [Git](https://git-scm.com/downloads)
- Install [Node.js >= 18](https://nodejs.org/)
- Configure your Git username and email to match the values below exactly, otherwise the commit history will not map to the correct contributor

---

Example: TJbubble

```bash
# Configure identity
git config --global user.name "TJbubble"
git config --global user.email "3268802588@qq.com"

# Clone the repository (choose one)
# Option A: GitHub SSH
git clone git@github.com:AlinaZLY/-AI-.git

# Option B: Gitee HTTPS
git clone https://gitee.com/chungeyydds/ai-campus-recruitment-platform.git

# Enter the project
cd -AI-

# Install dependencies
cd server && npm install && cd ..
cd client && npm install && cd ..
cd admin  && npm install && cd ..
```
